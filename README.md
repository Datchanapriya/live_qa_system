# Live VQA System

A real-time Visual Question Answering (VQA) system that combines Computer Vision and Natural Language Processing to answer questions based on images or live webcam input.

---

## Overview

The Live VQA System allows users to provide an image or live camera feed along with a natural language question. The system analyzes the visual content using AI models and generates relevant answers in real time.

This project demonstrates the integration of:
- Computer Vision
- Deep Learning
- NLP
- Real-time video processing

---

- Real-time webcam input

## Requirements

Make sure the following are installed:

- Python 3.9+
- pip
- OpenCV
- PyTorch
- Transformers Library

Install dependencies using:

```bash
pip install -r requirements.txt
```

---

## Installation

### Clone the Repository

```bash
git clone https://github.com/Datchanapriya/live_qa_system.git
cd live_qa_system
```

### Create Virtual Environment

```bash
python -m venv venv
```

### Activate Virtual Environment

#### Windows

```bash
venv\Scripts\activate
```

#### Linux / Mac

```bash
source venv/bin/activate
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

---

## Usage

Run the application using:

```bash
python main.py
```

After starting the application:
1. Open webcam or provide image input
2. Enter a question related to the image
3. The system generates an AI-based answer

---

## Example Questions

- What is in the image?
